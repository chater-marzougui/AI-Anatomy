## ğŸ›  Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chater-marzougui/AI-Anatomy.git
   cd AI-Anatomy
   ```

2. **Install dependencies**:
   ```bash
   flutter pub get
   ```

3. **Set up your Gemini API Key**:
    - Add your API key to main.dart:
      ```
      GEMINI_API_KEY=your_key_here
      ```

4. **Run the app**:
   ```bash
   flutter run
   ```
